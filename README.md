## Setup

- Clone this repo to an empty folder:
```
git clone https://github.com/Evva1023/Cookie-Factory-BE.git
```

- Open that folder in your IDE and install the dependencies by running:
```
npm install
```
- Once the dependencies are installed, run:
```
node index
```
- The server runs at localhost:8000
